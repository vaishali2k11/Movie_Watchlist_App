### <u>Run Commands on Frontend Side</u>
- Go Into movie_watchlist_app Directory
- cd movie_watchlist_app
- npm install
- npm start

It will open at "http://localhost:3000"

### <u>Run Commands on Server Slide</u>
- Go Into Server Folder Directory
- npm install
- npm server
